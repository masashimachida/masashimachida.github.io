    デスクトップからエアコンを操作する

    Nature Remo を使ってスマホからエアコンを操作しています（リモコンは諸事情により壊れました）。 Nature Remo っていうのはいわゆるスマートリモコンで、リモコンの赤外線信号を覚えさせてスマホから操作しようって製品です。 ただ、「ちょっと暑いな」と思って設定温度をさげようとする時なんかは、スマホを手に取り専用のアプリを立ち上げ、エアコンのメニューを選んで、微妙に使いづらいスライダーで１℃温度を下げる、という感じで一連の動きがとても煩わしいのです。 AmazonEcho も連携しているのでこいつに喋りかけてもいいんですが、電源ON/OFFならまだしも温度の上げ下げを伝えるのが割と難しい。というかもはや声を出すのもめんどうくさい。 そこで Nature Remo の API を使ってデスクトップ上から操作できるようにしました。 API はこちら。 Nature Inc. | Nature Developer Page 出来たものはこちら。masashimachida/remo-in-tray 起動するとメニューバーに現在の運転状況が表示される クリックしたところ 温度や風量はこう言う感じに設定する 作ってみたものの暖かくなってきたので出番があまりない。

    SLACK API の SOCKET MODE はイマイチ使いづらい

    apps.connections.open method | Slack ドキュメントにある通り https://slack.com/api/apps.connections.open に POST すると websocket のエンドポイントを教えてくれる訳だけど、CORS 制限があるからブラウザの js から叩けない。 リクエスト中に App-Level Tokens を含めるのでブラウザなんかで使うなよっていう話だとは思うんだけど、このためにサーバー立てるのはめんどくさいなー。 この socket 繋げればイベントがばんばん飛んでくるんだけど、認証通しているわけでもないので DM や private なチャンネルの情報は流れてこない。bot 用途なんだろうけどせっかく socket なんだからブラウザで使いたい。
