    エアコンを操作するのにリモコンを使うのがめんどうくさい

    • Fri, Apr 8, 2022
    • One-minute read

    リモコンと言ってますが、Nature Remo を使ってスマホからエアコンを操作しています（リモコンは諸事情により壊れました）。

    Nature Remo っていうのはいわゆるスマートリモコンで、リモコンの赤外線信号を覚えさせてスマホから操作しようって製品です。

    ただ、「ちょっと暑いな」と思って設定温度をさげようとする時なんかは、スマホを手に取り専用のアプリを立ち上げ、エアコンのメニューを選んで、微妙に使いづらいスライダーで１℃温度を下げる、という感じで一連の動きがとても煩わしいのです。 AmazonEcho も連携しているのでこいつに喋りかけてもいいんですが、電源ON/OFFならまだしも温度の上げ下げを伝えるのが割と難しい。というかもはや声を出すのもめんどうくさい。

    そこで Nature Remo の API を使ってデスクトップ上から操作できるようにしました。

    API はこちら。 Nature Inc. | Nature Developer Page
